# goog

A simple tool to search Google from the command line.

## Usage

```
Usage: goog [-n pages] [query]

Synopsis:
    -n [pages]                      Return n pages of results
    -h, --help                      Show this message
```

goog will color the output a little if STDOUT is the tty.
